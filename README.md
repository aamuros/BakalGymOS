# GymLedger

GymLedger is a web-based gym operations system for small local gyms.

This visual foundation phase includes:

- Next.js App Router with TypeScript and Tailwind CSS
- Simple landing page
- Visual-only login mockup
- Mobile-friendly app shell with sidebar and topbar
- Placeholder pages for the main GymLedger modules

This phase intentionally does not include database logic, authentication, entries, payments, shifts, exceptions, reconciliation, or GCash proof uploads.

## Pages

- `/`
- `/login`
- `/front-desk`
- `/owner-dashboard`
- `/members`
- `/payments`
- `/shifts`
- `/exceptions`
- `/reports`
- `/settings`
